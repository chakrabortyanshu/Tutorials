1. Download latest version from
       https://repo.maven.apache.org/maven2/org/projectlombok/lombok/

2. java -jar lombok-<version>.jar

3. patch Eclipse or STS that you are using

4. place the jar file in the folder <eclipse-setup>\plugins

5. Restart IDE.


6. if necessary, set argument bootclasspath as shown in the file eclipse.ini

	-Xbootclasspath/a:C:/eclipse-jee-oxygen-R/lombok.jar
